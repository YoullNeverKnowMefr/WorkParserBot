Put one image per category here. The file names must match categories.json.

Defaults referenced by categories.json:
  gamedev.jpg   -> 🎮 Game Dev
  cats.jpg      -> 🐱 Котовария
  it.jpg        -> 💻 IT / Backend
  design.jpg    -> 🎨 Design

If a category's image is missing, the post is still published (without that image)
and a warning is logged. To disable an image for a category, set "image": "" in
categories.json.
